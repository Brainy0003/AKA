Compiling a Console Application using the IDE

Open Visual C++ and go to File / New / Project

Explore the various Project Types and select Win32 Console Application.
Name your new project:
Hello
Next to Location, click the Browse button and navigate to the folder where you wish to create your first project. Visual C++ Probably created a folder for you in your My Documents folder called Visual Studio 2005, you can go to the folder in there called Projects if you like. Once you're inside your Projects folder, click the Open button.
You see that Visual C++ automatically made a Solution name for you which is the same as your project name. Fine.
Ensure that the Create Directory For Solution box is checked and click OK. This opens up the Application Wizard dialog.
Click Application Settings and check Console Application / Empty Project and click the Finish button. Visual C++ opens with your new empty project. To get started we can close the Start Page by clicking the little black X near the top right corner.
In the Solution Explorer you can see your project name followed by some empty folder icons. Right click on Source FIles and Add / New Item. Select C++ File (.cpp) and name the file Hello. Click the Add button and a new blank page opens called Hello.cpp.

Hit the F5 key and watch your program compile. When you see a message indicating that the compile was successful, hold the Ctrl key and hit F5.

A window pops up and displays your first program. Now you are a programmer. Big Whoop. Now get back to work.
