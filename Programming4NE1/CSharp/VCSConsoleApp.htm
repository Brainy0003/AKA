Create a Console Application using Visual C# Express 2005

OK, assuming you've already installed free Visual C#, launch the application:

The application opens with a default Start Page which you can close right away if you like.

Click on the New Project button


( or go to File / New Project )

This brings up a small box of predefined projects.

Select Console Application

you could go with the default name of ConsoleApplication1 or
double click on the name, type Hello and hit Enter.

Eventually a window opens with a default C# Console Application created for you.

Click one time under the word Main and hit Enter. Now type:
Co
a little popup window opens with a list of System Namespaces to choose from
you can use the up and down arrow keys to scroll through the list until you find the name
Console



Hit Enter and Visual C# completes the word for you
Now type a period followed by a W
scroll to the word WriteLine and hit enter
Now we type the rest of the line so it reads as follows:

Console.WriteLine ( "Hello YourName!" )

after a few seconds a little red squiggly line appears to tell us that the line of code is incomplete.

Add a semicolon ; at the end and it disappears.

So your finished code should look like this:

using System ;
using System.Collections.Generic ;
using System.Text ;       
namespace Hello
{
     class Program
     {
        static void Main ( string [ ] args )
        {
             Console.WriteLine ( "Hello YourName!" ) ; 
        }
     }
}

To compile your application, hit the F5 key
When your code finishes compiling, the application launches immediately, opens a console window displaying the word Hello and promptly exits, closing the console window. If you blinked you might have missed it.

To see your creation, hold Ctrl and hit F5
