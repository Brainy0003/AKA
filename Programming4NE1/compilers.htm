Free Compilers

To get started, first you will need a Compiler, an application which turns all your cryptic code into an even more indecipherable stream of 1s and 0s only your computer can understand.

One way to begin programming is to go out and buy an expensive compiler, then go out and buy a bunch of pricey books, then maybe throw some more money at an overpriced school. Another way is to get a free command line compiler and take advantage of reams of free code samples all over the internet. You need more time to think about it?

I tried the free Visual C++ download from Microsoft and found that none of the tutorials I could find on the web for Visual C++ were compatible, mostly because of changes to the interface. This is Visual C++ Express and not only were there no tutorials yet on the web, but a lot of the built in help feature had broken links, all the help seemed to assume you are an experienced programmer or merely migrating from Visual C++ 6 and was extremely technical and there was no "help for beginner's" feature, so although you can still compile your programs with Visual C++ O.K. from the command line, I ended up using Borland's free command line compiler first. Which worked fine.

If you want to get both the free Visual C++ and Visual C#, I think you may want to
install free Visual C++ Express first. Otherwise it may tell you to uninstall a bunch of other stuff. Then
install free Visual C# Express. Then if you intend to write Windows programs:
install the Microsoft Platform SDK ( Software Developer's Kit ).
Next if you want 3D or Graphics or Sound, you may want to
install DirectX SDK. If you want to do any Database programming, you may want to grab
free SQL Server. You may also need the .Net Framework runtime for who know's what, but it comes with the .Net Framework SDK.

With that little arsenal, you should be able to program just about anything. You will need to tell Visual C++ where to locate the SDKs, but we'll cross that bridge when we come to it. For now you should be totally set up to start programming simple applications which you can run from the command line.
