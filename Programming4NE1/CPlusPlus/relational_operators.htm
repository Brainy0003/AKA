Relational Operators

Let's say we want to determine whether it's morning, afternoon or evening based on the hour of day, between 1 and 24, for instance

 Hour_Of_Day = 11 ;

Now all we need to do is look at the value of Hour_Of_Day and if it is below a certain value, we know it's morning or if it's over another value, we know it's evening. This is accomplished with relational operators:

== ( Equal To )
< ( Less Than )
> ( Greater Than )
<= ( Less Than or Equal To )
>= ( Greater Than or Equal To )
!= ( Not Equal To)

Equal To:

Use = to assign a value to a variable:

My_Value = 7 ;

My_New_Variable = My_Variable ;

My_New_Variable = ( First_Variable + Second_Variable ) ;

Using == as a Conditional Statement:

 if My_Number == 7 do something
 if My_String == "hello" do something

Not Equal To:

Use != to denote that two values are not the same:

if User_Password != "Open_Sesame" 
cout << "Sorry, you do not have authorization"

Less Than:

 4 < 64537

Quite simply put: Whatever is on the smaller side of the symbol < is less than whatever's on the larger side.

Greater Than:

 96 > 12

If we said that 96 is greater than 12, we know that to be true, correct? I hope this isn't getting too complicated too fast.

So we can use if to make a conditional statement using < or >:

 if  ( 96 > 12 )
 cout << "96 is Greater Than 12" ;

This of course will print immediately since we know the condition to be true. It would be the same as saying:

 if  ( 1 = 1 )
 cout << "Brilliant Watson!" ;

However, if we said:

 if ( 1 > 12 ) 
 cout << "1 is not Greater Than 12" ;

it would print absolutely nothing, since we know for a fact that the conditional statement 1 > 12 is false.

That doesn't do a whole lot for us until we factor in Variables. Since we know that Variables can vary, we can set up a condition where if the Variable is above or below a certain number, it can trigger all kinds of events.

Right, so if we took the hour of the day, between 1 and 24, and wanted to determine whether it was A.M. or P.M. clearly we could say:

 if ( My_Variable < 12 ) 
 cout << "Good Morning" ; 
 if ( My_Variable > 12 ) 
 cout << "Good Evening" ;

That works. However, there is a small paradox. If it happens to be 12, this will return false since 12 is neither greater than nor less than 12. It is right on the money.
We could fix it this way using OR:

if ( (  My_Variable < 12 ) || ( My_Variable = 12 ) )
 cout << "Good Morning" ;

However there is a shorter way to do it, using <=

Less Than or Equal to

 if ( My_Variable <= 12 ) 
 cout << "Good Morning" ;

Now we are saying 12 or anything below it.

Greater Than or Equal to

 if ( My_Variable >= 18 ) 
 cout << "Good Morning" ;

Now we are saying 18 or anything higher.

Between two Numbers

O.K., next hitch is that Afternoon is in the middle. That means that it is anything over 12 AND under 18. We can use AND to ensure that it meets both conditions. Assuming we want 12 noon to be considered mid day:

 if (( Time_Of_Day >= 12 ) && ( Time_Of_Day < 18 )) 
 cout << "Good Afternoon" ;

So here would be the finished solution:

/*
 AmPm.cpp
	Calculate part of day based on Hour
*/
#include <iostream>
using namespace std ; 
int main ( void )
{
 cout << "Please enter the hour 0 - 24"
 cin >>  int Time_Of_Day
 
 if Time_Of_Day < 12 
 cout << "Good Morning" ; 

 if (( Time_Of_Day >= 12 ) && ( Time_Of_Day < 18 )) 
 cout << "Good Afternoon" 

 if ( Time_Of_Day >= 18 ) 
 cout << "Good Evening"

return 0 ;
}
