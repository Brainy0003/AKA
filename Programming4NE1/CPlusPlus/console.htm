Free C# Programming

OK, assuming you have already installed free Visual C++ Express from Microsoft you should now have the ability to author Console Applications, which you can run from the Command Line.
A Console Application is not something you run on a game console like the Xbox or something. Basically a Console Application is something which you would run on old school computers which didn't have Windows.

If you have installed the Microsoft Platform SDK, it should come with your free C# Command Line compiler.

If you installed Visual C++, you can skip the next part about adding the path. Visual C++ comes with it's own Command Prompt window which should be preconfigured for you.

Create a new Directory from the Command Prompt

Make a Directory to store your code in:
Type
md C++
and hit Enter. This creates a new folder called C++. Type dir to verify that the new directory is there.

Open specified directory

To open the C++ directory:
Type
cd c++
You will see the command prompt change to show that you are now in the C++ directory.

Inside that directory, let's make another directory for our first project called Hello:
Type
md Hello
then type dir to verify that the new Hello directory was created.

Open the Hello directory:
Type
cd hello

Create your first code

Type
notepad hello.cpp
Notepad opens and ask if it's OK to create a new file. Just hit Enter.
Type the following code:

Go to File / Save As
Save the file hello.cpp in the Hello folder you created earlier located at My Computer / Hard Drive ( C: / C++ / Hello ). From the Command Prompt, you should now be able to type dir and see your newly created file: hello.cs

Compile the code

To compile your code: from the Command Prompt type:
cl hello.cpp
If all went well your program should compile without any errors. You should find the newly created program called hello.exe in the Hello folder. Type dir and you can see your new program. To run it, type
hello
The program executes and prints the message to the Command Prompt window. Nothing spectacular, but congratulations! you're a programmer. Big whoop. Now get back to work.

Note that once your have entered a few commands into the Command Prompt window, you can hit the Up or Down Arrow keys to recall any commands you already entered. So once you have compiled one program, if you need to compile it a second time, you just arrow up till you find the command from the previous compile and hit Enter. Now hit the Arrow Down key and there's the name of the file. Hit Enter to run the program again. Saves typing.
