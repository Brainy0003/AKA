Loops using C++

Consider this:

cout << "10" << endl ;
cout << "9" << endl ;
cout << "8" << endl ;
cout << "7" << endl ;
cout << "6" << endl ;
cout << "5" << endl ;
cout << "4" << endl ;
cout << "3" << endl ;
cout << "2" << endl ;
cout << "1" << endl ;

That seems like a lot of work for a simple thing like a countdown doesn't it? After all, we're starting with a number, printing it, subtracting 1, printing it again, etc.. Seems like that should be a job for the computer to be doing.

While Loop

Run the following program:

/* While.cpp 
   Countdown from constant number to 1 
   Written by: Dillinger
   Date: 12/08/05
*/
#include <iostream>
using namespace std;       
int main ( void )
         {
         int Counter = 10;
         while ( Counter >= 1 )
           {
             cout << Counter ;
             --Counter ;
             cout << endl ;
           }
 cout << "Blastoff!" ;
return 0;
} /* main */

Basically the structure of a While Loop looks like this:

while ( expression = true ) do something

or

while ( expression = true )
{
do something
do something
do something
}

As long as the expression is true, it will repeat it's task over and over. Once the expression becomes false, it will continue on it's merry way. In this case our expression contains a variable:
( Counter >= 1 )
Every time the While loop cycles through, it will decrease that variable by 1.
--My_Variable decreases the value in increments of 1.
++My_Variable increases the value in increments of 1.
When Counter reaches a value of 0, the expression becomes false: Counter is no longer greater than or equal to 1, and the program will break out of the loop immediately and continue executing whatever follows. The Counter value of 0 never gets printed.

Let's try counting up from 1 to 10:

/* WhileUp.cpp 
   Countdown from constant number to 1
   Written by: Dillinger
   Date: 12/08/05 
*/

using namespace std;       
int main ( void )
         {
         int Counter = 1;
         while ( Counter <= 10 )
           {
             cout << Counter ;
             ++Counter ;
             cout << endl ;
           }
 cout << "Blastoff!" ;
return 0;

Do While

/* DoWhile.cpp
   Add a list of integers.
   Keep going until Expression = True
   Written by: Dillinger
   Date: 12/10/05
*/       
#include <iostream>
using namespace std ; 
int main ( void )
{
/* Local Declarations */
         int User_Input ;
         int sum = 0 ;
/* Statements */
         cout << "Enter some numbers ( 0 to stop ) : " << endl ;
     do
       {
         cin >> User_Input ; 
         sum += User_Input ;
         cout << "Total: "<< sum << endl ;
       }
      while ( User_Input != 0 ) ;
         cout << endl << "Program terminated " <<endl ;
         cout << "Have a day. :|" << endl ; 

return 0 ;
} /* main */

In a nutshell, This will continue to DO { over & over } While ( Expression = True )
When ( Expression = False ) exit do loop
and continue

For Loops 

/* ForLoop.cpp
   Print decreasing numbers from 10 to 1
   Written by: Dillinger
   Date: 12/08/05
*/       
#include <iostream>
using namespace std ;
int main ( void )
{
      int Counter = 10 ;
      for ( Counter ; Counter >= 1 ; Counter-- )
         cout << Counter << endl ;
         cout << "Blastoff!" ; 
return 0 ;
} /* main */

The basic For Loop is constructed like this:

for ( StartingValue ; EndingValue ; Increase/Decrease )
      do something

or you can loop two or more commands within curly brackets:

for ( StartingValue ; EndingValue ; Increase/Decrease    )
{
   do something
   do something else
   do something etc.
}

In the above example we declared and assigned the value of Counter at the top. We could just as easily have declared it within the For Loop:

for ( int Counter = 10; Counter >= 1 ; Counter-- )
