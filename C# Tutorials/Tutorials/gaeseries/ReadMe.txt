MathMatic
=========
Learn and Drill Multiplication Tables

Installation
------------
Unzip the ZIP archive to a temporary folder
and double-click "SetUp.exe" to start the
installation wizard.

If you have a late version of WinZip, you
can double-click "SetUp.exe" directly from
within WinZip's interface.

Default installation folder:
"Program files/MathMatic"

Uninstall
---------
Double-click "Uninstall MathMatic.exe" located in
MathMatic's directory.

MathMatic "Junior Edition" is freeware.
it never expires and has no nag screens.

Enjoy!
